## 📂 我的造物集 (The Artifacts)

### 1. 🧘 Focus (桌面应用)
> **起因：** 市面上的 To-Do 软件要么太臃肿，要么有广。我就想要个干干净净、能让我沉浸下来的小工具。

这这是一款完全用 Python 原生开发的桌面应用，但它一点也不像传统的 Python GUI 软件。

我“逼着”AI 抛弃了老旧的原生控件，用 Canvas 重新绘制了所有的复选框和滑块，就为了实现我心目中的现代感和熔岩橙配色。

**它最酷的地方：**
* **沉浸感拉满**：没有标题栏，窗口半透明，就像浮在桌面上的水印。
* **颜值即正义**：精心调配的深色/浅色模式，看着就舒服。
* **极简且私密**：没有乱七八糟的联网功能，数据就存在本地 JSON 里，秒启动。

👉 **试试看：** 确保你有 Python 环境，下载 `Focus.py` 直接运行就行。此外，我使用Nuitka打包成exe文件存放于Releases。

### 2. 🤖 Gemini Chat Navigator (浏览器脚本)
> **起因：** Gemini 的长对话太难用了！每次想翻之前的提问都得疯狂滚动鼠标，极其痛苦。我需要一个目录，立刻，马上。

这是个标准的“自己动手，丰衣足食”的产物。我花了大概 15 分钟和 AI check 需求，就搞定了这个油猴脚本。

我跟 AI 说：“我要在右边加个侧边栏目录，样式要现在流行的毛玻璃效果，而且切换对话时它得自动刷新。” 然后，它就真的做出来了，还完美绕过了 Google 复杂的安全限制。

**它最酷的地方：**
* **自动生成目录**：再也不用大海捞针找聊天记录了。
* **毛玻璃 UI**：不仅好用，还得好看，放在页面上毫无违和感。
* **无感运行**：完美适配了 Gemini 的单页应用模式，切换对话时自动更新。

👉 **安装：** 装好 [Tampermonkey](https://www.tampermonkey.net/) 扩展，点击仓库里的 `gemini-toc.user.js`，点“Raw”直接安装。

### 3. 💘 Douyin Auto-DM Bot (自动化脚本)
> **起因：** 和某人的抖音火花断了，决定搞个每天定时发消息的自动化脚本。

这是一个基于 **Selenium** 的浏览器自动化脚本，但它比普通的爬虫“聪明”得多。

在开发过程中，我跟 AI 经历了数次迭代。我们解决的最棘手的问题是：**怎么骗过现代前端框架？** 抖音的输入框是 React 写的，直接注入文字会导致发送按钮无法点亮。于是我让 AI 学会了“假动作”——输入文字后，模拟按下空格再删除，强行激活发送按钮。

**它最酷的地方：**
* **React 破防术**：内置了专门针对现代前端框架的输入逻辑，模拟真人键盘敲击，由不得发送按钮不亮。
* **极速冲浪模式**：它不会傻傻地发完一个人就关浏览器重开，而是像真人一样，发完一个，直接在顶部搜索栏搜下一个，行云流水。
* **智能守门员**：Cookie 过期了弹登录框？它不会报错崩溃，而是会自动暂停，并在控制台喊你：“主人请扫码！”等你扫完，它继续干活。
* **强迫症级日志**：强力屏蔽了 Selenium 底层那些烦人的 USB 报错和 SSL 握手失败红字，控制台输出只有你关心的“发送成功”。

👉 **食用方法：** 1. 确保你有 Chrome 或 Edge 浏览器及对应驱动。
2. 在代码的 `TARGET_USERS` 列表里填上你朋友的抖音号和想说的话（支持自动计算纪念日天数）。
3. 运行脚本，首次扫码登录，之后就是全自动了。
